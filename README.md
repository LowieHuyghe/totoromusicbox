# totoromusicplayer

## Setup
```
# Support git
sudo apt-get install -y git
# generate ssh
ssh-keygen
# make sure you can connect to pi
sudo systemctl start ssh
# add public key to repo
# clone repo
git clone git@github.com:LowieHuyghe/totoromusicplayer.git
# twice
./bin/install/i2samp.sh
# install
./install.sh
```
